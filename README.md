# My Create React App

## Technologies

### Framework

React

### Package Bundler

Vite

### Linting and code style

ESLint (Google rule set)
Prettier

### Syling

TailwindCSS

### Package Manager

Yarn

## How to use

1. Install dependencies:
   > yarn
2. Run in dev mode:
   > yarn dev
3. Build project:
   > yarn build
4. Preview built project:
   > yarn preview
